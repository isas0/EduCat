# EduCat
EduCat è una piattaforma web di prenotazione e acquisto di lezioni private rivolta a tutor di diverse discipline, studenti di qualsiasi grado di istruzione e genitori.

L’obiettivo del sistema è offrire a studenti e genitori (in caso di studenti minorenni) un modo semplice per confrontare i tutor in base a modalità di insegnamento, fascia di prezzo e
disponibilità, così da prenotare lezioni secondo le proprie esigenze e possibilità economiche. 

I tutor potranno registrarsi sulla piattaforma e offrire ripetizioni a pagamento. 

Tecnologie utilizzate:
- Eclipse
- Server Apache Tomcat
- JUnit 5
- Mockito
- Selenium Web Driver
